# Vision

## Product name
GeoScore

## Tagline
SEO & GEO Analyzer for sites and apps

## Core idea
GeoScore analyzes a public page and returns a serious, explainable score for SEO and GEO readiness.

## Main promise
Paste a URL. Get a serious SEO & GEO score.

## Problem solved
Many sites are visually polished but weak in:
- technical readiness
- content structure
- clarity of value proposition
- AI-search readability
- citation readiness
- trust and entity signals

GeoScore turns these weak points into:
- scores
- issues
- priority fixes

## Product philosophy
Simple on the surface, serious underneath.

## V1
- analyze one public URL
- detect page type
- compute global score
- compute SEO score
- compute GEO score
- surface issues and fixes
- show confidence and limitations
- allow rescan
- allow public report sharing

## V2
- multi-page analysis
- site-level scoring
- historical comparison
- competitor comparison

## V3
- app listing analysis
- ASO + GEO scoring
- rewrite suggestions
- scheduled scans
