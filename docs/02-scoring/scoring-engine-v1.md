# Scoring engine V1

## Goal
Turn one normalized page extraction into:
- global score
- SEO score
- GEO score
- issues
- recommendations
- confidence

## Score model
- Global Score = 50 percent SEO + 50 percent GEO

## SEO pillars
- technical basics
- on-page structure
- content signals
- discoverability

## GEO pillars
- clarity of intent
- extractability
- citation readiness
- entity trust

## Core rule
Deterministic rules are primary.
LLM is secondary and bounded.

## Required outputs
- scores
- sub-scores
- issues
- recommendations
- confidence
- limitations
